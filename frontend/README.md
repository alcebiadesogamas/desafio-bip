# Frontend Angular

Aplicacao frontend Angular para consumir os endpoints criados no backend-module.

a arquitetura proposta e a seguinte:

- Arquitetura baseada em componentes
  - pasta src/app/core contém services e models para realizar operacoes no backend.
  - pasta src/app/pages contém os componentes de pagina

para executar a aplicação:
```
git clone https://github.com/alcebiadesogamas/desafio-bip.git
cd desafio-bip.git
npm install
ng serve
```

Operações:
  - 1 - listagem de benefícios (Pagina principal)
  - 2 - cadastro de benefícios (Pagina de cadastro)
  - 3 - edição de benefícios (Pagina de edicao)
  - 4 - exclusão de benefícios (Modal de exclusao)
  - 5 - transferência de saldo (Pagina de transferencia)

Versão do Angular - 21+
